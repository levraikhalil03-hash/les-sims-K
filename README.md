# les-sims-K

Prototype documentaire d'un **jeu de simulation de vie original** inspiré du genre life sim, avec :

- un document de design complet ;
- un système de création de personnage ;
- un mode construction ;
- une proposition d'architecture technique pour un prototype.

## Contenu

- `docs/original-life-sim-design.md` : document principal de conception.

## Positionnement

Ce projet décrit un **jeu original** : il s'inspire des grands principes des jeux de simulation de vie, mais évite de reproduire à l'identique l'identité, l'interface, les assets ou les systèmes d'un jeu commercial existant.
